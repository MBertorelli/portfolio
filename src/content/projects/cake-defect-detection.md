## The problem

Defective products on a bakery production line have to be caught while the
line is moving — manual inspection doesn't scale with line speed and fatigues
into misses.

## The approach

A focused, single-purpose vision pipeline: acquisition, detection and decision
logic driven by a YAML configuration so thresholds and line-specific
parameters can be tuned on site without touching code. Built deliberately
small — a production line is not the place for a framework.

## Outcome

The line got automated visual inspection, and the collaboration proved out:
this first project grew into an ongoing freelance relationship covering
further inspection systems — screen edge detection and real-time mesh defect
segmentation.
